# MultitaskCoder — Static Site

Pure static website. No build step required.

## Structure
```
/
├── index.html
├── style.css
├── script.js
├── manifest.webmanifest
├── sw.js
├── .nojekyll
└── assets/
    ├── images/
    ├── icons/  (favicon.ico, icon-192.png, icon-512.png)
    ├── fonts/
    └── videos/
```

## Deploy to GitHub Pages
1. Push these files to the root (or `/docs`) of your repo.
2. Settings → Pages → Deploy from branch.
3. Done.

Open `index.html` directly to preview locally.
